# UsefulScripts

## Installation
```bash
$ ./install.sh
```

## How To Use
* To execute designated command. Below is an example to get information about weather.
```bash
$ my-command weather
天気     : 曇のち晴
最低気温 : 9℃ 
最高気温 : 17℃
```

* To know what command there is.
```bash
$ my-command list
weather   get infomation about weather and temperature
```
